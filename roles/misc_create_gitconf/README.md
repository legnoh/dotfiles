# misc_create_gitconf

The tiny role for create git config file into `~/.config/git` and include into `.gitconfig`.

## Requirements

None

## Role Variables

| Variable                  | Required | Default | Choices                                   | Comments |
|---------------------------|----------|---------|-------------------------------------------|----------|
| `misc_create_gitconf_src` | true     | -       | target filename in `files` or `templates` | -        |

## Dependencies

None
